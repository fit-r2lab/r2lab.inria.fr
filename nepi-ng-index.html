nepi-ng

NEPI new-generation is a tool for running and orchestrating network experiments that involve several tens of target hosts.

Check it out

Look at what nepi-ng scripts look like on the R2lab website, and in particular in this simple tutorial in the context of that platform.

The guts

At this point it only supports interacting with ssh, and so it's made of 2 simple python libraries:

the asynciojobs orchestration engine the apssh ssh layer
