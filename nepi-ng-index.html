nepi-ng

NEPI new-generation is a tool for running and orchestrating network experiments that involve several tens of target hosts.

Check it out

The R2lab website provides a simple tutorial in the context of that platform.

The pieces

nepi-ng is built right on top of the asyncio library from python-3.5, which makes it very efficient. At this point it only supports interacting with ssh, and so it's made of 2 simple python libraries:

asynciojobs, the orchestration engine apssh, the ssh layer
